## Usage

To start the project, open a terminal window and launch make:

```bash
$ make
```

Then you can launch the executable with some arguments:

```bash
$ ./flow [-f <file>] [-c <netflow_collector>[:port]] [-a <active_timer>] [-i <inactive_timer>] [-m <count>]
```
